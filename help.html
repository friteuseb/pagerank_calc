Help - PageRank Calculator

PageRank is an algorithm used by Google to assess the relative importance of a webpage based on the number and quality of links pointing to it.

How does the PageRank calculation work?

The basic formula for calculating the PageRank of a page P is as follows:

PR(P) = (1 - d) / N + d * Σ (PR(Pi) / L(Pi))

  • PR(P) : PageRank of page P
  • d : Damping factor (usually set to 0.85)
  • N : Total number of pages
  • Σ : Sum for all pages Pi pointing to P
  • PR(Pi) : PageRank of page Pi
  • L(Pi) : Number of outbound links from page Pi

The damping factor d represents the probability that a user will continue clicking on links, rather than starting a new search from a new page. This factor is generally set to 0.85.

Example of Calculation

Suppose we have three pages: A, B, and C. Here is a simplified example calculation:

  1. Initialization: Suppose each page starts with a PageRank of 1.0.
  2. Calculation after one iteration:

Page A:
PR(A) = (1 - 0.85) / 3 + 0.85 * (PR(C) / 1) = 0.05 + 0.85 * 1 = 0.9

Page B:
PR(B) = (1 - 0.85) / 3 + 0.85 * (PR(A) / 1) = 0.05 + 0.85 * 1 = 0.9

Page C:
PR(C) = (1 - 0.85) / 3 + 0.85 * (PR(B) / 1 + PR(A) / 1) = 0.05 + 0.85 * (1 + 1) = 1.8

These calculations are repeated several times until the values converge.
