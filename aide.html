Aide - Calculateur de PageRank

Le PageRank est un algorithme utilisé par Google pour évaluer l'importance relative d'une page web en fonction du nombre et de la qualité des liens pointant vers elle.

Comment fonctionne le calcul du PageRank ?

La formule de base pour calculer le PageRank d'une page P est la suivante :

PR(P) = (1 - d) / N + d * Σ (PR(Pi) / L(Pi))

  • PR(P) : PageRank de la page P
  • d : Facteur d'amortissement (généralement fixé à 0.85)
  • N : Nombre total de pages
  • Σ : Somme pour toutes les pages Pi pointant vers P
  • PR(Pi) : PageRank de la page Pi
  • L(Pi) : Nombre de liens sortants de la page Pi

Le facteur d'amortissement d représente la probabilité qu'un utilisateur continue de cliquer sur les liens, plutôt que de commencer une nouvelle recherche à partir d'une nouvelle page. Ce facteur est généralement fixé à 0.85.

Exemple de calcul

Supposons que nous ayons trois pages : A, B, et C. Voici un exemple de calcul simplifié :

  1. Initialisation : Supposons que chaque page commence avec un PageRank de 1.0.
  2. Calcul après une itération :

Page A :
PR(A) = (1 - 0.85) / 3 + 0.85 * (PR(C) / 1) = 0.05 + 0.85 * 1 = 0.9

Page B :
PR(B) = (1 - 0.85) / 3 + 0.85 * (PR(A) / 1) = 0.05 + 0.85 * 1 = 0.9

Page C :
PR(C) = (1 - 0.85) / 3 + 0.85 * (PR(B) / 1 + PR(A) / 1) = 0.05 + 0.85 * (1 + 1) = 1.8

Ces calculs sont répétés plusieurs fois jusqu'à ce que les valeurs convergent.
